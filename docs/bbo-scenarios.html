Browser extension · bridgebase.com

Practice specific bidding conventions on BBO

BBO already supports practice bidding tables, but setting them up for a specific convention is slow and cumbersome. This extension adds a PBS panel to BBO's sidebar — one click to configure a table for any scenario in your collection.

One click to set up a practice table

Choose a bidding scenario from the PBS panel and the extension configures a BBO practice table automatically — no manual setup required.

Scenarios organized by convention

Dozens of scenarios covering notrump ranges, transfers, Stayman, slam conventions, competitive bidding, and more.

Optional BBA comparison

After each hand, see how the BBA bidding engine would have bid it — with differences highlighted and a full double-dummy analysis.

Only activates on bridgebase.com — no permissions on any other site. Adds a PBS tab to BBO's existing right-side panel.
